{% docs unemployment_rate %}

Percentage of the labor force that is unemployed, seasonally adjusted.  
Covers civilian noninstitutional population aged 16+.  
Monthly frequency, sourced from BLS via FRED.  
See https://fred.stlouisfed.org/series/UNRATE for full methodology.

{% enddocs %}

{% docs labor_force_participation_rate %}

Percentage of the population that is working or actively looking for work, seasonally adjusted.  
Covers civilian noninstitutional population aged 16+.  
Monthly frequency, sourced from BLS via FRED.  
See https://fred.stlouisfed.org/series/CIVPART for full methodology.

{% enddocs %}

{% docs consumer_price_index %}

Price index of a basket of goods and services paid by urban consumers, seasonally adjusted.  
Monthly frequency, sourced from BLS via FRED.  
See https://fred.stlouisfed.org/series/CPIAUCSL for full methodology.

{% enddocs %}

{% docs real_gdp_growth_rate %}

Annual growth rate of the real gross domestic product, seasonally adjusted.  
Quarterly frequency, sourced from BEA via FRED.  
See https://fred.stlouisfed.org/series/A191RL1Q225SBEA for full methodology.

{% enddocs %}

{% docs federal_funds_effective_rate %}

Weighted average rate of overnight lending of federal funds between banks.  
Steered by the Federal Reserve's Interest on Reserve Balances rate towards the target range.  
Monthly frequency, sourced from the Federal Reserve Board via FRED.  
See https://fred.stlouisfed.org/series/FEDFUNDS for full methodology.

{% enddocs %}

{% docs real_median_household_income %}

Real median household income in the U.S.  
Annual frequency, sourced from the U.S. Census Bureau via FRED.  
See https://fred.stlouisfed.org/series/MEHOINUSA672N for full methodology.

{% enddocs %}
